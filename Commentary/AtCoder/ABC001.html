A - 積雪深差

問題文

ある時刻の積雪深H1とその1時間前の積雪深H2が与えられます。この時、この1時間の積雪深差H1 - H2の値を計算して出力してください。

解説

入力として２つの整数H1,H2が与えられるので、H1-H2を計算し、出力すれば良い。 

          #include<iostream>
using namespace std;

int main() {
    int H1, H2;
    cin >> H1 >> H2;
    cout << H1 - H2 << endl;
    return 0;
}
